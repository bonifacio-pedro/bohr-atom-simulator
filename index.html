Note: Electrosphere layers radii and rotation speed are scaled for visualization. Real radius ~0.0529 nm (Bohr Radius) and speed ~10⁶ m/s cannot be animated directly. K-shell radius is multiplied by 10; other layers use factor 5. Speed is reduced by 10⁻⁸. The atom is precisely a representation, the calculations are based on Bohr's work with the Hydrogen atom, it would be very complex to apply each pattern of radius, speed and color dispersion to each existing element.
